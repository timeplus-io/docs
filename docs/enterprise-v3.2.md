## Releases
We recommend using stable releases for production deployment. Engineering builds are available for testing and evaluation purposes.

### 3.2.2 {#3_2_2}
Released on 04-09-2026. Installation options:
* For Linux or Mac users: `curl https://install.timeplus.com/3.0 | sh` [Downloads](/release-downloads#3_2_2)
* For Docker users (not recommended for production): `docker run -p 8000:8000 docker.timeplus.com/timeplus/timeplus-enterprise:3.2.2`
* For Kubernetes users: `helm install timeplus/timeplus-enterprise --version 11.0.6`

Component versions:
* timeplusd 3.2.2
* timeplus_appserver 3.2.0
* timeplus_connector 3.2.0
* timeplus cli 3.0.0
* timeplus byoc 1.0.0
