# 函数

Timeplus 支持 ANSI-SQL 标准语法。 以下功能适用于各种使用案例。 如果您需要更多功能，请联系我们。

:::info

请注意，Timeplus 中的函数区分大小写且始终为小写。 例如， 您可以运行 `SELECT count(*) FROM car_live_data` 而不是 `SELECT COUNT(*) FROM car_live_data` 一些关键词，例如 `SELECT` 或 `FROM` 是不区分大小写的。

:::
