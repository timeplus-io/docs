Timeplus是一个简单、强大且经济的流处理平台。 它有不同的可用方式：

- [Timeplus Proton](proton): the core engine open-sourced at [GitHub](https://github.com/timeplus-io/proton) under Apache 2.0 License. 它是一个快速而轻量级的流式SQL引擎。
- [Timeplus Enterprise](timeplus-enterprise): the production-ready commercial product, with 2 deployment options:
  - Cloud: a fully-managed unified platform for streaming and historical data processing.
  - Self-hosted: same feature sets as Timeplus Cloud, able to run and scale everywhere, from edge to your data center. Customize according to your configuration needs.

Timeplus 提供强大的端到端功能，帮助数据团队快速、直观地处理流和历史数据，各种规模和行业的组织都可以访问。 它使数据工程师和平台工程师能够使用 SQL 解锁流数据价值。

Timeplus 控制台允许轻松连接到不同的数据源（例如Apache Kafka、Confluent Cloud、Redpanda、NATS、Web Socket/SSE、CSV文件上传等），通过SQL查询探索流媒体模式，向其他系统或个人发送实时见解和警报，以及创建仪表板和可视化效果。

Still curious about [the benefits of using Timeplus](why-timeplus)? Check out the [showcases](showcases) to see how Timeplus customers use our unified streaming and historical processing platform.
