Timeplus是一个简单、强大且经济的流处理平台。 它有不同的可用方式：

- [Timeplus Proton]（Proton），在 Apache 2.0 许可下在 [GitHub] (https://github.com/timeplus-io/proton) 上开源的核心引擎。 它是一个快速而轻量级的流式SQL引擎。
- [Timeplus Enterprise]（timeplus-Enterprise）是一款可用于生产的商用产品，有 2 种部署选项：
  - 云：完全托管的统一平台流媒体和历史数据处理。
  - 自托管：与 Timeplus Cloud 相同的功能集。 从边缘到数据中心，可在任何地方运行和扩展。

Timeplus 提供强大的端到端功能，帮助数据团队快速、直观地处理流和历史数据，各种规模和行业的组织都可以访问。 它使数据工程师和平台工程师能够使用 SQL 解锁流数据价值。

Timeplus 控制台允许轻松连接到不同的数据源（例如Apache Kafka、Confluent Cloud、Redpanda、NATS、Web Socket/SSE、CSV文件上传等），通过SQL查询探索流媒体模式，向其他系统或个人发送实时见解和警报，以及创建仪表板和可视化效果。
