# Integration with Grafana

For self-hosted Timeplus Enterprise or Timeplus Proton, you can also try [the Grafana plugin for Timeplus](https://grafana.com/grafana/plugins/timeplus-proton-datasource/). 源代码位于 https://github.com/timeplus-io/proton-grafana-source。 你可以使用该插件运行 SQL 流，并在 Grafana 中构建实时图表，而无需刷新仪表板。 查看 [此处](https://github.com/timeplus-io/proton/tree/develop/examples/grafana) 获取示例设置。
